# Brain Stroke Prediction

A Machine Learning based web application that predicts the likelihood of a brain stroke based on patient health information.

## Features

- Stroke risk prediction
- Interactive Streamlit UI
- Machine Learning model deployment
- Real-time prediction

## Dataset

Kaggle Playground Series S3E2 Stroke Dataset

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- Joblib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
